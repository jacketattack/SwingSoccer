## Setup for Mac to run locally
### Install Dependencies
1. Install NPM and Node if you do not already have them
    1. `brew update`
    2. `brew install node`
2. Simple http server for hosting files with live updates
- `npm -g install browser-sync`

### Start
- `npm start`
